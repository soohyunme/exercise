# 개인 맞춤형 운동 루틴 추천 서비스

#### 서비스 개요
- 국민체력측정 처방 정보를 분석하여 개인별 신체정보와 식단에 따른 **맞춤 운동 루틴 추천**
- 처방 운동 동영상 가이드 연결을 통해 운동에 익숙하지 않은 사용자도 **올바른 운동 자세 학습 가능**
- **운동 기록과 성취도를 평가**하고 달력을 통해 운동 기록을 한눈에 확인
